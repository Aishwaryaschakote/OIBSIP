# Auralis - Landing Page

This project is a responsive landing page developed as part of the **Web Development & Designing Internship** at **Oasis Infobyte**.

The landing page is designed for a fictional productivity platform called **Auralis**. It focuses on creating a clean and modern user interface using only HTML and CSS.

## Features

- Sticky navigation bar
- Hero section with call-to-action buttons
- Features section
- How It Works section
- Testimonials section
- Footer with contact information and social links
- Responsive design for desktop, tablet, and mobile devices

## Technologies Used

- HTML5
- CSS3
- CSS Flexbox
- CSS Grid

## Folder Structure

```
WebDev-L1-LandingPage/
│── assets/
│   ├── images/
│   └── icons/

│── style.css

```

## How to Run

1. Download or clone this repository.
2. Open the `WebDev-L1-LandingPage` folder.
3. Open `index.html` in your browser or run it using VS Code Live Server.
